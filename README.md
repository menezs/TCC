## Instalação

Instale esse projeto com pip

```bash
  git clone 
```
```bash
  cd back-front
```
```bash
  python -m venv venv
```
```bash
  .\.venv\Scripts\activate
```
```bash
  pip install -r requirements.txt
```
```bash
  python app.py
```
No seu navegador de acesse o link: [localhost:5001](http://localhost:5001/)
